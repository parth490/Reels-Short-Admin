About

🎬 Welcome to the Admin Panel

Powering Entertainment with Coins!

A smart admin panel to manage reels, movies, and episodes with a coin-based access system. Users can unlock content or earn free coins by watching ads. This admin panel is the command center for managing a dynamic content platform featuring reels, movies, and episodes. At the heart of this ecosystem is an engaging coin-based access model that adds value for users while giving you full control over content and user interactions.

💰 How the Coin System Works :

Users are assigned a specific number of coins in their account. These coins act as virtual currency that can be used to unlock and watch premium content—whether it's a trending reel, a blockbuster movie, or an exclusive episode.

📺 Unlock with Ads :

To make the experience more accessible, users can also earn coins by watching ads, promoting user engagement while creating monetization opportunities.

🔧 Admin Features :

  • Manage all uploaded content : reels, movies, and episodes.
  • Monitor and adjust user coin balances.
  • Track ad engagement and rewarded coins.
  • View detailed analytics on content performance and user behavior.

This panel puts all the power in your hands—manage, monetize, and grow your content platform while offering users an interactive, rewarding viewing experience.

Features

🎞️ Manage Content

Upload, organize, and update reels, movies, and episodes directly from your dashboard.

💰 Coin System

Control how users spend or earn coins to unlock premium content.

📺 Ad-Based Rewards

Users can watch ads and receive free coins, encouraging more engagement.

📊 User Analytics

Monitor watch time, series views, and coin activity for better content decisions.

🛠️ Access Control

Set coin requirements per video and manage who can view what.

🔐 Role Permissions

Manage the entire panel through a single admin account with full control, as no additional roles or access levels are available in the system.

Hardware Requirements

Local Backend Installation

  1. Navigate to /backend directory using command prompt.
  2. Install Node.js (If already installed then no need to install)
  3. Run npm install to install dependencies.
  4. Create database From Localhost Panel. URL look like http://localhost/phpmyadmin
  5. Select created database and check import option in top panel and click on that
  6. Click on Choose File and select database.sql File
  7. Set database credentials, base URLs and secrets in .env.
  8. Use npm start to start local server
  9. Ensure MySQL running in local environment.

Local Website Installation

  1. Navigate to /frontend directory using command prompt.
  2. Install Node.js (If already installed then no need to install)
  3. Run npm install to install dependencies.
  4. Edit .env to set API base URL.
  5. Use npm start to start local dev server.
  6. Open http://localhost:3000 in browser.

Local Admin Installation

  1. Navigate to /admin directory using command prompt.
  2. Install Node.js (If already installed then no need to install)
  3. Run npm install to install dependencies.
  4. Edit .env to set API base URL.
  5. Use npm start to start local dev server.
  6. Open http://localhost:3001 in browser.

VPS Installation

1. Prerequisites

  • Access to a VPS with root or sudo privileges (e.g., Ubuntu 24.04).
  • Domain name pointed to your server (optional for testing).
  • Basic understanding of Linux commands.

2. CloudPanel Installation Steps

  • Step 1: Update your system
    sudo apt update && sudo apt upgrade -y
  • Step 2: Install required packages
    sudo apt install curl sudo -y
  • Step 3: Run CloudPanel installer (For Ubuntu 22.04 / 24.04)
    curl -sSL https://installer.cloudpanel.io/ce/v2/install.sh | sudo bash
  • Step 4: Access CloudPanel (Once installation is complete, open your browser and go to)
    https://your-server-ip:8443 Follow the web-based setup wizard to create your admin account.

Project Installation on VPS

1. Backend Installation

  1. Navigate to the backend directory: cd /var/www/drama-shorts/backend
  2. Install dependencies: npm install
  3. Create and configure the .env file (DB, JWT, PORT, etc.).
  4. Run database migrations or seeders if needed.
  5. Start the server with PM2: pm2 start index.js --name drama-backend
  6. Check logs to confirm the server is running: pm2 logs drama-backend

2. Frontend Installation

  1. Navigate to the frontend directory: cd /var/www/drama-shorts/frontend
  2. Install dependencies: npm install
  3. Update the .env file with the production API URL.
  4. Build the production app: npm run build
  5. Serve with a process manager (e.g., pm2) : pm2 serve dist 3000 --spa
  6. Ensure it's proxied correctly via Nginx on port 3000.

3. Admin Installation

  1. Navigate to the frontend directory: cd /var/www/drama-shorts/frontend
  2. Install dependencies: npm install
  3. Update the .env file with the production API URL.
  4. Build the production app: npm run build
  5. Serve with a process manager (e.g., pm2) : pm2 serve dist 3000 --spa
  6. Ensure it's proxied correctly via Nginx on port 3000.
